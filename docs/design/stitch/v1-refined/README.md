# VoteReady India — Stitch UI Screens v1 Refined

## Purpose

This folder contains the refined Google Stitch UI/UX output for VoteReady India.

These screens are the visual design reference for the first implementation pass of the VoteReady India web app.

## Product Context

Product Name: VoteReady India  
Tagline: Ask. Understand. Be vote-ready.

VoteReady India is an assistant-first, source-backed civic education product for India’s first-time and soon-to-be voters.

The assistant is the product. Guided journeys, timelines, saved guidance, reminders, source cards, and settings are supporting layers.

## Folder Contents

- `screens/` — refined screen images exported from Stitch
- `raw-export/` — original Stitch export ZIP for traceability
- `DESIGN.md` — implementation guidance for Antigravity
- `UI_COPY_SAFETY_WATCHLIST.md` — text/copy issues that must be fixed during implementation

## Implementation Rule

These screens are design references only.

Do not copy unsafe placeholder procedural claims, dates, or source freshness labels directly into production UI.

All procedural or deadline-sensitive guidance must be generated through the approved source-backed assistant flow.
